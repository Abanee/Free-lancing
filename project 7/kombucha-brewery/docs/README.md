## Features

- **Artisanal Design**: Custom-curated color palette and typography.
- **Dark Mode**: Toggleable dark mode for enhanced viewing.
- **RTL Support**: Built-in support for right-to-left languages.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Premium Imagery**: High-quality brewery visuals.

## Setup

Simply open `index.html` in your browser to view the landing page.
